Segunda Atividade em JavaScript – Concatenação de Campos

Descrição

Projeto introdutório em JavaScript que utiliza um formulário HTML para capturar nome e sobrenome, concatenar os valores e exibir uma mensagem de saudação personalizada.

Objetivo

Trabalhar com formulários HTML

Capturar dados usando document.form

Concatenar strings em JavaScript

Exibir mensagens com window.alert()

Tecnologias

HTML

JavaScript

Funcionamento

O usuário digita o nome e o sobrenome

Clica no botão Concatenar

O sistema exibe um alerta com a mensagem: “Olá Nome Sobrenome”
